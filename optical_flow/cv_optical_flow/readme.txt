Our Presentation has 3 parts:

1. Basic idea2. Correlation-based optical flow3. Differential-based and Dense optical flow

correlation.py is correlation-based optical flow (for 2)

differencial.py is differencial-based optical flow (for 3)

timer.py is just for estimate performance

camera_demo/opencv_demo_point_track.py is demo point track (Please click the window and add some point to track when you use this demo) (for 1)

camera_demo/opencv_demo_extract_bg.py is demo of catch object from background (for 3)

camera_demo/opencv_demogetpeople.py is demo of tracking moving object (for 3)


(python version is 2.7.11, opencv version is 2.4.12_2)
